## 📥 New User Profile Inputs

Users now provide 4 key pieces of information:

1. **Current Job Role** - e.g., "Marketing Executive"
2. **Job Domain** - 16 predefined domains (Business, Tech, Healthcare, etc.)
3. **Position Level** - Entry, Mid-Level, Senior, Lead, Manager, Director, Executive
4. **Current Salary** - Optional, used for salary progression
