# Job Market AI - Frontend

A modern, high-performance React dashboard built with Vite, Tailwind CSS, and Framer Motion.

## ✨ Highlights

- **Dynamic Transitions**: Instant UI switching between Landing Page and Dashboard upon authentication.
- **Shared Auth State**: React Context-based authentication for seamless state management.
- **Data Visualization**: Interactive Recharts components for market trends.
- **Micro-animations**: Smooth, premium feel powered by Framer Motion.

## 🏗️ Architecture

- `src/auth`: Context provider and custom hooks for authentication.
- `src/components/features`: Core modules like Roadmap, Recommendations, and Trends.
- `src/services/api.js`: Axios configuration for backend communication.

## 🚀 Running Locally

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.
